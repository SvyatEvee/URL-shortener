О проекте

Как часто у вас возникает потребность сохранить сайт для быстрого доступа, но при этом совсем не хочется засорять закладки или главную страницу браузера? Этот сервис позволяет создавать короткие ссылки и пользоваться ими с любого устройства, где вы авторизованы.

Стек технологий

Go Postgres REST API gRPC JWT Nginx Docker Docker Compose GitHub Actions JS SSL/TLS
